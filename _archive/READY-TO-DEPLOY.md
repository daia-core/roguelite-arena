## Deploy to Vercel

**Web UI (2 minutes):**
1. Visit https://vercel.com/new
2. Import `daia-core/roguelite-arena`
3. Root: `frontend`
4. Framework: Vite  
5. Deploy

**CLI (30 seconds):**
```bash
cd /workspace/work/roguelite-game/frontend
npx vercel --prod
```

## How to Play

- Desktop: WASD + Space (dash) + E (blast)
- Mobile: Touch joystick + ability buttons
- No login - single player, auto-saves in browser
